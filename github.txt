🜁 LI: A Narrative-Driven Host Enumerator
A three‑act metamorphic horror engine disguised as a system profiler.

🧩 What Is This Project?
LI is a narrative‑driven, persona‑shifting system enumerator.
It begins as a simple host profiler, but its behavior, personality, and instability evolve as it collects information from the machine it runs on.

Act I (the only act implemented so far) is a theatrical system enumerator that:

profiles the host

logs evidence

reacts emotionally to what it finds

escalates its persona based on “closeness”

slips into instability based on hotwords, mood, and collected data

persists its state between runs

Future acts (not implemented yet) will introduce:

a GUI for reviewing and deleting collected data

a final “rebirth” phase where LI becomes a new personality shaped by user choices

But for now, the focus is Act I: the facehugger.
